# CRUDUsuariosWeb

Proyecto de aplicación web CRUD para gestión de usuarios.  
Desarrollado en Java con Servlets, JSP y Maven.  

## Instalación
1. Clonar el repositorio: `git clone https://github.com/pachodavid1988/CRUDUsuariosWeb.git`
2. Importar el proyecto en IntelliJ IDEA o Eclipse.
3. Configurar Tomcat en el IDE.
4. Ejecutar el servidor y probar en `http://localhost:8080/CRUDUsuariosWeb`.

## Requisitos
- Java 17+
- Apache Tomcat 9+
- Maven

## Ejecución
1. Clonar el repositorio.
2. Compilar con Maven.
3. Desplegar en Tomcat.
4. Acceder a `http://localhost:8080/CRUDUsuariosWeb`.
